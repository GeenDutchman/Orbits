This program simulates a neutron star orbiting a binary black hole system

python3 binaryNewton.py

--star				Flag to begin changing star parameters
	--x0,	-x		Sets the x coordinate position of the star
	--y0,	-y		Sets the y coordinate position of the star
	--z0,	-z		Sets the z coordinate position of the star
	--vx0,	-vx		Sets the Vx velocity component of the star
	--vy0,	-vy		Sets the Vy velocity component of the star
	--vz0,	-vz		Sets the Vz velocity component of the star
--tstep,-ts			Sets the time step for the simulation data
--tmax, -tm			Sets the maximum run time for the simulation data
--default, -d			Shows the default parameters

--record, -r			Prints the initial conditions as a comment
